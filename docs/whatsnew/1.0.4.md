# What's New — v1.0.4

## ✨ 新功能

- **评论定时通知**：Review 模块新增「定时通知」，每天到点自动批量拉取新增差评并推送到 Telegram，不用手动巡检。
  - 改为后端原生线程触发，即使窗口最小化 / 退到后台也能准点执行。
  - 支持按回复状态筛选（只推未回复 / 全部），已通知的评论不会重复打扰。
  - 新增「复查提醒」：顺带扫出「已回复后又被用户更新」的评论，避免漏跟进。
  - 巡检完成后自动刷新评论列表页，无需手动刷新。
- **评论按账号隔离**：评论 / Play 本地存储按 Google 账号分开保存，多账号之间互不串数据。
- **Android 版本显示**：评论里的 API Level 直接显示为对应的 Android 版本，更直观。
